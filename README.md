# Duck Duck Tutor
Duck Duck Tutor (DDT) leverages large language model (LLM) agents to provide an interactive code-tutoring service for students. Students are able to ask DDT to:
* Explain Python fundamentals,
* Debug code snippets,
* Provide example code with explanations,
* Generate practice exercises and evaluate answers,
* Encourage learning through interactive feedback.

Additionally, users may choose to interact with DDT's multi-agent architecture. This system leverages multiple specialized agents capable of discussion and collaboration, improving the learning experience of users.
